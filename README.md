# Proyecto-modelo-deteccion-fallas-VE
Modelo predictivo que anticipa fallas en vehículos eléctricos usando datos históricos de telemetría (temperaturas, ciclos de carga, RPM, torque y desgaste). Se aplica limpieza de datos, EDA y XGBoost para clasificar eventos de falla (DTC) y apoyar mantenimiento preventivo reduciendo riesgos y tiempos de inactividad.
